St. Patrick Homebanking API

Repository Main page Documentation

> /account/auth - POST

Send a valid email and password - responds with sent email & access token (stp-token).

body: {
"email": "example@mail.com",
"password": "************"
} Content-type: "application/json" response: {
"email": "example@mail.com",
"token": "****************"
} || { "error": error }

> /account/find - POST

Send a valid email and token - responds with Account object.

body: { "email": "example@mail.com" } header: { "stp-token": "***************" } Content-type: "application/json" response: { Account } || { "error": error }

> /account/transfer - POST

Make a money transfer between two valid accounts. Origin and destiny refers to the CCI code.

body: {
"origin": "000000000000000",
"destiny": "111111111111111",
"motive": "insert example motive"
"amount": > 0
} header: { "stp-token": "***************" } Content-type: "application/json" response: { "status": "success" }|| { "error": error }

> /user/find - POST

Send a user's DNI & valid token - responds with User object.

body: {
"dni": "12345678",
} header: { "stp-token": "***************" } Content-type: "application/json" response: { User }|| { "error": error }

> /transaction/find - POST

Send a valid transaction ID & token - responds with Transaction object.

body: {
"transactionId": >= 0,
} header: { "stp-token": "***************" } Content-type: "application/json" response: { Transaction }|| { "error": error }

> /transactions/account - POST

Send a valid account CCI & token - responds with all Transactions having this account as the origin or destiny.

body: {
"cci": "111111111111111",
} header: { "stp-token": "***************" } Content-type: "application/json" response: { [ Transactions ] }|| { "error": error }
